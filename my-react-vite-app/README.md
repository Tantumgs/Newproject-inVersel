# My React Vite App

This is a simple React application created using Vite. It serves as a demonstration of how to set up a React project with Vite and includes a custom `<App>` component.

## Project Structure

```
my-react-vite-app
├── index.html          # Main HTML document
├── package.json        # npm configuration file
├── vite.config.js      # Vite configuration settings
├── .gitignore          # Git ignore file
├── src
│   ├── main.jsx        # Entry point of the React application
│   ├── App.jsx         # Custom App component
│   ├── index.css       # CSS styles for the application
│   └── assets          # Directory for static assets
└── README.md           # Project documentation
```

## Getting Started

To get started with this project, follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/your-username/my-react-vite-app.git
   ```

2. Navigate to the project directory:
   ```
   cd my-react-vite-app
   ```

3. Install the dependencies:
   ```
   npm install
   ```

4. Start the development server:
   ```
   npm run dev
   ```

5. Open your browser and go to `http://localhost:3000` to see the application in action.

## Features

- Custom `<App>` component with a main title and description of React.
- Footer that dynamically displays the current year.
- Built with Vite for fast development and optimized builds.
